# Overview

A highly scable basic ecommerce rest API which includes basic authentication using http cookie.

Some features:

- express session to store authentication session
- sequelize orm to connect db
- initial db entries to demostire the endpoints

## Enviroment and requirements

- [NodeJS version 12](https://nodejs.org/en/) or higher needs to be installed
- Install mysql. Default configuration uses mysql connection
- Update src/config file with your db connection
- run <code>npm run install</code> to install all dependencies

## Available Scripts

### `npm start`

Runs the server on port [http://localhost:5000](http://localhost:5000)
